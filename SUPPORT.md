# Support

## Where to ask for what

- Questions and launch feedback:
  [platformkit-community discussions](https://github.com/septagon-dev/platformkit-community/discussions)
- Bugs in the flagship product surface:
  [platformkit issues](https://github.com/septagon-dev/platformkit/issues)
- Security problems:
  [SECURITY.md](SECURITY.md)

## Before opening an issue

- Check the roadmap in [ROADMAP.md](ROADMAP.md).
- Check the product contract in [docs/PRODUCT_CONTRACT.md](docs/PRODUCT_CONTRACT.md).
- Make sure the problem belongs in this flagship repository and not in an owning
  split repository.

## Support expectations

This is an open-source project. Best-effort community and maintainer support is
provided, but no response-time guarantee is implied.
